# 🏏 IPL Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes IPL data from 2008–2025 and provides interactive insights into team performance, player statistics, and season trends.

---

## Dashboard Features

- Interactive Dashboard
- KPI Cards
- Dynamic Filters
- Season Analysis
- Team Analysis
- Player Analysis
- Performance Tracking
- Data Visualization

---

## Key Insights

- Team Performance Comparison
- Player Performance Analysis
- Season-wise Trends
- Match Statistics
- Interactive Reports

---

## Tools Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Modeling
- Microsoft Excel
